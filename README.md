# SOLITECH Monitoring

Project ini merupakan aplikasi monitoring yang saat ini **masih bersifat visual (tampilan dashboard saja)** dan **masih dalam tahap pengembangan**.

Project ini dibuat untuk kebutuhan **mata kuliah Interaksi Manusia dan Komputer (IMK)**.

## Teknologi yang Digunakan

- **React** (library utama frontend)
- **React DOM** (rendering komponen ke browser)
- **React Router DOM** (navigasi/routing halaman)
- **Vite** (build tool dan development server)
- **ESLint** (linting dan menjaga kualitas kode)
